# 🌞 3D Solar Eclipse Simulation

A Computer Graphics project developed using Python and OpenGL that simulates a **3D Solar Eclipse** with realistic visual effects.

## 📌 Project Overview

This project demonstrates the movement of celestial bodies in a 3D space environment.
It visualizes the **Sun, Earth, and Moon** and shows how a solar eclipse occurs when the Moon comes between the Sun and Earth.

The project also includes advanced computer graphics concepts such as:

* **Fog Effect** 🌫️
* **Blending Effect (Transparency)** 🪟
* **Shading / Lighting** 💡
* **3D Animation** 🔄
* **Keyboard Interaction** ⌨️

## ✨ Features

✅ 3D Sun model with glow effect
✅ Earth revolving around the Sun
✅ Moon orbiting around Earth
✅ Fog effect for depth perception
✅ Transparent glow using blending
✅ Realistic lighting and shading
✅ Camera zoom control using keyboard
✅ Smooth animation loop
✅ 500×500 simulation window

## 🛠️ Technologies Used

* Python
* PyOpenGL
* GLUT
* OpenGL Graphics API

## 🎮 Controls

| Key | Action   |
| --- | -------- |
| W   | Zoom In  |
| S   | Zoom Out |

This project implements:

2. Animation

## 🚀 How to Run

Install required libraries:

pip install PyOpenGL PyOpenGL_accelerate

Then run:

python main.py

## 🎯 Project Objective

The main objective of this project is to visualize a **solar eclipse simulation in 3D** while applying advanced computer graphics effects and animation techniques.
